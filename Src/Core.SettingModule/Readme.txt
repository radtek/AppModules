Менеджер настроек сохраняет\считывает настройки из источника, определяемого свойством ReadWriteProvider.

- Сохранению\восстановлению подлежат все открытые свойства экземпляра переданного методам LoadAllSettings\SaveAllSettings;
- Если запись с указанным именем в источнике отсутствует или имеет пустое значение, свойству присваиваются следующее значение:
  - для ссылочных типов - значение null;
  - для значений - значение по-умолчанию
- Если в течение инициализации поля происходит исключение, процесс прекращается и уже инициализированные поля не откатываются. Соответственно 
  ответствунность на сохранение целостности экземпляра класса настроек ложится на вызывающий код (предпочтительно передавать менеджеру копию класса)

  Поддерживаемые типы:
  - встроенные элементарные типы и их Nullable-варианты:
	  string
	  Int16 
	  Int32 
	  Int64 
	  float 
	  double
	  decimal
	  bool
	  DateTime
	  TimeSpan
	  Size
	  Point
	  Color
	  Byte
- ссылочные тип:
	  Font
	  byte[]
- обобщенный список List из любых вышеуказанных типов;
    

  Атрибуты:

- IgnoreAttribute - при сохранении\восстановлении свойство игнорируется
- NullToEmptyAttribute - в случае, если  запись с указанным именем в источнике отсутствует или имеет пустое значение, свойству присваиваются следующее значение:
    - результат вызова контруктора по-умолчанию, если такой есть (есть у всех типов-значений);
	- строкам - ""
	- если нет конструктора по-умолчанию - null;
	- Nullable - типам - null;
- NullNotChangedAttribute - в случае, если  запись с указанным именем в источнике отсутствует или имеет пустое значение, значение свойства не изменяется;
- NullToDefaultAttribute - в случае, если  запись с указанным именем в источнике отсутствует или имеет пустое значение, полю присваивается значение указанное в атрибуте.
  При этом формат значения должен соответствовать формату TypeDescriptor.GetConverter(propertyType).ConvertFromString при настройках локализации "EN-US";
